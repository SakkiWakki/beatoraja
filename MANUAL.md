# beatoraja Manual

## Global Keys

| Key | Action |
|-----|--------|
| F1 | Toggle FPS display |
| F2 | Update song folder |
| F3 | Open explorer with selected song |
| Ctrl+F3 | Copy song MD5 hash |
| Ctrl+Shift+F3 | Copy song SHA256 hash |
| F4 | Toggle fullscreen/windowed |
| F6 | Save screenshot |
| F7 | Post to Twitter |
| F8 | Add favorite song |
| F9 | Add favorite chart |
| F10 | Autoplay folder |
| F11 | Open IR (Internet Ranking) |
| F12 | Open skin configuration |

## Song Select

| Key | Action |
|-----|--------|
| UP/DOWN | Scroll song list |
| LEFT | Close folder |
| RIGHT/ENTER | Open folder or play song |
| ESCAPE | Exit application |
| 0 | Search songs |
| 1 | Toggle key mode filter |
| 2 | Toggle sort order |
| 3 | Toggle LN mode |
| 4 | Change replay slot |
| 5 | Open detailed options (hold) |
| 6 | Open configuration |
| 7 | Toggle rival data |
| 8 | Show songs in same folder |
| 9 | Open help |
| Scroll Wheel | Quick scroll |

### With START held (Play Options)

| Input | Action |
|-------|--------|
| Lane keys (left side) | Decrease 1P option |
| Lane keys (right side) | Increase 1P option |
| Scratch keys | Adjust gauge type |
| Additional keys | Adjust hi-speed |

### With SELECT held (Assist Options)

Toggles custom judge window, constant speed, judge area display, legacy note, mark processed notes, BPM guide, and mine note mode.

### With START+SELECT held

Adjusts notes display timing, BGA display, and gauge auto shift.

## Gameplay

| Key | Action |
|-----|--------|
| ESCAPE | Stop and return to song select |
| UP/DOWN | Adjust lane cover |
| Scroll Wheel | Quick adjust lane cover |

### With START held

| Input | Action |
|-------|--------|
| Lane keys (left) | Decrease hi-speed |
| Lane keys (right) | Increase hi-speed |
| Scratch keys | Adjust lane cover size |

### With SELECT held

| Input | Action |
|-------|--------|
| Lane keys (left) | Decrease note duration |
| Lane keys (right) | Increase note duration |

### START+SELECT

Toggle between lane cover, lift, and hidden modes. Hold for ~2 seconds to exit play.

### Autoplay/Replay Only

| Key | Action |
|-----|--------|
| 1 | 25% speed |
| 2 | 50% speed |
| 3 | 200% speed |
| 4 | 300% speed |

## Result Screen

| Key | Action |
|-----|--------|
| ENTER/ESCAPE | Return to song select |
| 1-4 | Save replay to slot 1-4 |
| F11 | Open IR |
| Scroll Wheel | Navigate ranking list |

Chart key actions vary by mode:
- First few lane keys: proceed to next song
- Upper lane keys: replay with same/different options
- Scratch key: change graph type

## Key Configuration

| Key | Action |
|-----|--------|
| LEFT/RIGHT | Switch game mode (5K, 7K, 9K, 10K, 14K, 24K) |
| UP/DOWN | Navigate key list |
| ENTER | Enter key assignment mode |
| DEL | Clear key assignment |
| ESCAPE | Save and exit |
| 1 | Switch music select input mode |
| 2 | Change controller device 1 |
| 3 | Change controller device 2 |
| 7 | Reset all to default (keyboard) |
| 8 | Reset all to default (controller) |
| 9 | Reset all to default (MIDI) |

## Skin Configuration

| Key | Action |
|-----|--------|
| Scroll Wheel | Navigate options |
| ESCAPE | Save and exit |

## Default Keyboard Layout

```
Lane 1-5:  Z  S  X  D  C
Scratch:   F (left)  V (right)
Start:     Left Shift / Right Shift
Select:    Left Ctrl / Right Ctrl
```
